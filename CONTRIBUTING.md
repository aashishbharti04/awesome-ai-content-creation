# Contributing

Thanks for helping keep this list awesome! 🎉

## Adding a tool
1. Fork the repository
2. Create a branch: `git checkout -b add/tool-name`
3. Add the tool to the **correct category table** in `README.md`
4. Use this format: `| **Name** | one honest line on what it's best for | [domain](https://link) |`
5. Open a pull request

## Guidelines
- The tool must be **real, working, and relevant** to content creation
- Keep descriptions short, honest, and hype-free
- Use the **official** website link
- One tool per pull request keeps reviews easy

Following the [Awesome List](https://github.com/sindresorhus/awesome) spirit — quality over quantity. 🙌
